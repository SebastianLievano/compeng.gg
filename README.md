# CompEng.gg

A site for learning coding.

## Frontend

Setting up

```
cd frontend
npm i
```

Running

```
cd frontend
npm run dev
```

## Backend

Setting up

```
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -U pip
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Running

```
cd backend
source venv/bin/activate
python manage.py runserver
```
